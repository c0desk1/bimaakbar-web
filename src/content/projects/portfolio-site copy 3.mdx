## Tentang Proyek

Website ini dirancang untuk menampilkan semua aspek kreatif saya: musik, blog, dan konten video.

## Fitur

- Halaman rilisan musik terintegrasi dengan Blogger API
- Embed video YouTube otomatis
- CMS mini via Cloudflare Workers

## Teknologi yang Digunakan

- Astro sebagai framework frontend
- Tailwind CSS untuk styling
- Cloudflare Workers untuk backend ringan dan caching
